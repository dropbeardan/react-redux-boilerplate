# React-Redux Boilerplate

Boilerplate code for a React app served by NodeJS and Webpack-Dev-Server.

## Stack

1. React (Client DOM)
1. Redux (REACT Pattern)
1. LESS (CSS)
1. Axios (HTTP Request)
1. React/Redux Router (Routing)
1. Babel (Transpiler)
1. Webpack (Packager, Builder)
1. Jest (Test)
